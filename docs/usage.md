# 사용법

## 1. 참가자 명단 만들기
화면 상단의 "참가자 명단" 입력창에 이름을 입력하고 "적용" 버튼을 누릅니다.

- **여러 명을 한 번에 입력**: 이름을 공백으로 구분해서 붙여넣으세요.
  예) `철수 영희 민수 지훈` → 4명(철수, 영희, 민수, 지훈)으로 등록됩니다.
- **글자 수로 인원만 빠르게 맞추고 싶을 때**: 공백 없이 아무 글자나 입력하면 그 글자 수만큼 참가자가 생성됩니다.
  예) `12345678` (8글자) → 참가자 8명, 이름은 각각 "1","2","3"...으로 채워집니다.
  주의: 실제 이름 하나(예: `홍길동`)를 공백 없이 붙여넣으면 3글자로 쪼개져 3명이 되니, 적용 후 뜨는
  안내 문구("이름 N명으로 인식했습니다" / "글자 단위로 N명 인식했습니다")를 확인하세요.
- 100명을 초과해서 입력하면 자동으로 100명까지만 반영되고 잘렸다는 안내가 표시됩니다.

## 2. 개별 이름 수정/삭제
"참가자 명단" 탭에서 각 참가자 칸에 직접 타이핑해서 이름을 바꿀 수 있고, 각 줄의 ✕ 버튼으로 한 명씩 삭제할 수
있습니다. 빈 칸으로 둔 참가자는 실행 시 자동으로 "참가자 N"이라는 이름으로 채워집니다.

## 3. 조 나누기 실행
하단의 "🔀 무작위로 그룹 나누기" 버튼을 누르면:
1. 전체 명단을 무작위로 섞고
2. 3~4명 단위로 조를 나눈 뒤 (5명일 경우만 예외적으로 5명이 한 조)
3. "결과" 탭으로 자동 전환되어 조 편성 결과를 보여줍니다.

같은 명단으로 다시 섞고 싶다면 버튼을 다시 누르면 됩니다 (매번 새로운 무작위 결과).

## 4. 전체 초기화
화면 제목 옆의 "🔄 다시하기" 버튼을 누르면 입력했던 명단, 결과, 붙여넣기 입력값이 모두 지워지고
기본 상태(참가자 8명, 빈 이름)로 되돌아갑니다.

## 5. 휴대폰 앱처럼 설치하기
1. 모바일 브라우저(Chrome/Safari 등)로 라이브 주소(https://tarrence-lee.github.io/group-maker/)를 엽니다.
2. 브라우저 메뉴에서 "홈 화면에 추가"(Android: 자동으로 설치 안내가 뜨기도 함, iOS Safari: 공유 버튼 →
   "홈 화면에 추가")를 선택합니다.
3. 홈 화면에 생긴 아이콘으로 실행하면 브라우저 주소창 없이 전체화면 앱처럼 열립니다.
4. 한 번 접속한 뒤에는 서비스 워커가 앱 화면을 캐시해두므로, 오프라인 상태에서도 다시 열 수 있습니다
   (단, 앱 자체가 외부 데이터 없이 동작하므로 오프라인에서도 모든 기능이 그대로 동작합니다).
