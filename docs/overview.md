# 오징어골프 그룹나누기 — 프로젝트 개요

## 이 앱은 무엇인가
참가자 명단을 입력하면 무작위로 섞어서 3~4명 단위의 조로 나눠주는 단일 페이지 웹 앱입니다.
서버나 빌드 도구 없이 `index.html` 하나로 동작하며, 모바일 홈 화면에 설치해 앱처럼 실행할 수 있는
PWA(Progressive Web App)로 구성되어 있습니다.

## 배포 주소
- 라이브 앱: https://group-maker-five.vercel.app/
- 저장소: https://github.com/tarrence-lee/group-maker

## 핵심 기능
1. **참가자 명단 입력** — 상단 입력창에 이름을 붙여넣고 "적용"을 누르면 자동으로 참가자 목록이 채워집니다.
   - 공백으로 구분된 문자열(예: `철수 영희 민수`) → 각 단어를 참가자 이름으로 인식
   - 공백이 없는 문자열(예: `가나다라마`) → 글자 하나하나를 참가자로 인식 (한 명의 실제 이름을 붙여넣을 때는
     의도치 않게 글자 단위로 쪼개질 수 있으니 적용 후 상태 메시지로 인식 결과를 꼭 확인하세요)
   - 100명 초과 시 100명까지만 반영되고 안내 메시지가 표시됩니다.
2. **개별 참가자 편집** — 참가자 명단 탭에서 각 이름을 직접 수정하거나, 행마다 있는 ✕ 버튼으로 개별 삭제 가능
3. **무작위 조 편성** — 하단 "🔀 무작위로 그룹 나누기" 버튼을 누르면 Fisher–Yates 셔플 후 3~4명 단위로 조를
   나눕니다. 5명일 때는 3+2로 쪼갤 수 없으므로 예외적으로 5명이 한 조가 됩니다.
4. **다시하기(전체 초기화)** — 제목 옆 🔄 버튼으로 참가자 명단·입력값·결과를 모두 초기화하고 기본 상태(8명 빈 칸)로
   되돌립니다.
5. **모바일 최적화** — 기본 8명 기준으로 대부분의 모바일 화면(375×667 이상)에서 스크롤 없이 한 화면에 표시되도록
   설계되었습니다.

## 기술 스택 및 제약
- 순수 HTML/CSS/바닐라 JavaScript, 빌드 도구·프레임워크·백엔드 없음
- API 키나 외부 네트워크 호출 없음 (전부 클라이언트 로컬 동작)
- WCAG 2.1 AA 접근성 기준(키보드 조작, 색상 대비, ARIA 라벨, 44×44px 터치 타겟) 준수
- Vercel로 정적 호스팅 (GitHub 연동, push 시 자동 배포)
