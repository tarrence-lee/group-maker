# 오징어골프 그룹나누기

참가자 명단을 무작위로 섞어 3~4명 단위 조로 나눠주는 모바일 친화적 PWA입니다.

- **라이브 앱**: https://group-maker-five.vercel.app/
- **문서**: [docs/overview.md](docs/overview.md) · [사용법](docs/usage.md) · [배포](docs/deployment.md) · [변경 이력](docs/changelog.md)

빌드 도구나 백엔드 없이 순수 HTML/CSS/JS로 동작하며, 모바일 브라우저에서 "홈 화면에 추가"로 설치해
앱처럼 사용할 수 있습니다.
